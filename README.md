# 📄 PDFuse — PDF Combiner

A lightweight **browser-based PDF combiner** that lets users merge multiple PDF files into a single document instantly.

⚡ Fast • 🔒 Private • 💻 Runs entirely in your browser

---

## 🚀 Features

- 📂 **Drag & Drop Upload**
  - Easily add PDF files by dragging them into the drop zone.

- 🔀 **Reorder PDFs**
  - Change the order of documents before merging using drag-and-drop.

- ➕ **Add More Files**
  - Add additional PDFs anytime without resetting the list.

- 🗑 **Remove or Clear Files**
  - Remove individual files or clear the entire list.

- ✏ **Custom Output Name**
  - Choose the filename before downloading the merged PDF.

- 📊 **Progress Indicator**
  - Visual progress bar while PDFs are being processed.

- 🔒 **100% Client-Side Processing**
  - No files are uploaded. Everything happens locally in your browser.

---

## 🛠 Built With

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **pdf-lib** (PDF manipulation library)

CDN used:

```
https://cdnjs.cloudflare.com/ajax/libs/pdf-lib/1.17.1/pdf-lib.min.js
```

---

## 📦 How It Works

1. Upload or drag multiple PDF files.
2. Files appear in a sortable list.
3. Rearrange them using drag-and-drop.
4. Click **Combine & Download**.
5. Enter the desired filename.
6. The merged PDF downloads automatically.
